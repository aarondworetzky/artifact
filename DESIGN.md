Generated by /design-consultation on 2026-03-19.

## Product Context

- **What this is:** A local-first Python CLI tool that turns ChatGPT export JSON into a personal intelligence system — semantic clustering, search, personality profiling, and Obsidian export.
- **Who it's for:** Technical founders, heavy ChatGPT users, Obsidian enthusiasts, people who think in text. Ben is the archetype: felt the pain acutely, built a manual workaround.
- **Space/industry:** Personal knowledge management (PKM) / productivity tools. Peers: Bear, Obsidian, Rewind.
- **Project type:** CLI app now → web upload product in ~6 weeks (per strategy doc).

---

## Aesthetic Direction

- **Direction:** Considered Minimal
- **Decoration level:** Intentional — typography does 90% of the work. The remaining 10% is a single warm accent color and subtle grain texture on dark surfaces (3–5% opacity SVG noise filter).
- **Mood:** Clean and clinical like Apple Notes × Notion, with an archival warmth that neither Obsidian nor Bear has. Feels like a library, not a dashboard. Says something is worth reading.
- **What makes it ours:** No purple (Obsidian), no teal (Bear). Amber/copper is archaeological and ownable. Fraunces serif for display is a deliberate risk — nobody in this space uses a serif for UI. It gives artifact. a literary point of view.
- **Reference sites researched:** bear.app (warm, notebook-y), obsidian.md (dark, developer-forward, purple)

---

## Typography

- **Display / Hero:** Fraunces (variable, 9–144 optical size, weight 300–600)
  - Used for screen titles, Profile archetype lines ("You're a Late-Night Builder."), Your Story headlines, and any moment that needs a voice. The variable optical size means it reads beautifully at both 48px hero and 24px section heading.
  - Italic weight in Fraunces is exceptional — use it for the key insight word in each archetype line.
  - Loading: `https://fonts.googleapis.com/css2?family=Fraunces:ital,opsz,wght@0,9..144,300..900;1,9..144,300..900&display=swap`

- **Body / UI:** Instrument Sans (weight 300–600)
  - Clean geometric sans with excellent weight differentiation on dark backgrounds (addresses DM Sans's known contrast weakness). The workhorse for all body text, menu labels, descriptions, and UI chrome.
  - Loading: `https://fonts.googleapis.com/css2?family=Instrument+Sans:ital,wght@0,300..700;1,300..600&display=swap`

- **Data / Code:** Geist Mono (weight 300–500, tabular-nums)
  - All statistics, scores, percentages, dates, counts, and code snippets. The precision font. `font-variant-numeric: tabular-nums` everywhere numbers appear.
  - Loading: `https://fonts.googleapis.com/css2?family=Geist+Mono:wght@300..500&display=swap`

- **Scale (8-point base):**
  ```
  xs    12px / 1.5   — metadata, captions, mono labels
  sm    14px / 1.6   — secondary body, sidebar items
  base  16px / 1.7   — primary body text
  lg    18px / 1.65  — intro paragraphs, slightly elevated body
  xl    24px / 1.4   — section subheadings
  2xl   32px / 1.25  — h2, screen titles
  3xl   48px / 1.1   — hero display (Fraunces)
  ```

## Icons

- **CLI layer (rich terminal):** Emoji — terminal-native, familiar, no alternative. Keep the current set: 📖 💡 🔍 📊 🪞 📤 ⚙️ etc.
- **Web / GUI layer (future):** Lucide Icons — outline style, 1.5px stroke, 24px grid. Perfectly matches Considered Minimal aesthetic. No filled icons, no colored circles, no icon-in-badge patterns.
  - CDN: `https://unpkg.com/lucide@latest`
  - Key icons: `book-open`, `lightbulb`, `search`, `bar-chart-2`, `user`, `upload`, `settings`

## Color

- **Approach:** Restrained — one accent, warm neutrals, color is rare and meaningful.

### Dark mode (default)

```
Background:       #141412   — warm near-black, not cold #000
Background-2:     #1C1B18   — cards, panels
Background-3:     #242320   — sidebar, code blocks, inset surfaces
Border:           #2E2C28   — subtle dividers
Text primary:     #C8C3BA   — warm parchment (not stark white)
Text secondary:   #8A8580   — secondary content, descriptions
Text tertiary:    #5C5954   — dim labels, placeholders, metadata
Accent:           #C17F3E   — amber/copper (all interactive elements, highlights, key stats)
Accent dim:       #7A5128   — hover states, pressed states
```

### Light mode

```
Background:       #FAFAF8   — warm off-white
Background-2:     #F3F1EC
Background-3:     #ECEAE4
Border:           #D8D5CE
Text primary:     #1A1918   — warm near-black
Text secondary:   #6B6762
Text tertiary:    #9B9893
Accent:           #C17F3E   — same across modes
Accent dim:       #E8C99A
```

### Semantic colors

```
Success:          #4A7C59   — confirmations, export complete, good signal
Warning:          #C17F3E   — same as accent (intentional: warnings should be noticed, not alarming)
Error:            #9B3A3A   — failures, missing data

```

### Grain texture (dark mode only)

Apply a 3–5% opacity SVG noise filter to all dark background surfaces. This makes dark mode feel archival and physical rather than like a developer terminal.

```css
body::before {
  content: '';
  position: fixed;
  inset: 0;
  background-image: url("data:image/svg+xml,%3Csvg viewBox='0 0 200 200' xmlns='http://www.w3.org/2000/svg'%3E%3Cfilter id='n'%3E%3CfeTurbulence type='fractalNoise' baseFrequency='0.9' numOctaves='4' stitchTiles='stitch'/%3E%3C/filter%3E%3Crect width='100%25' height='100%25' filter='url(%23n)' opacity='1'/%3E%3C/svg%3E");
  background-size: 200px 200px;
  opacity: 0.03;
  pointer-events: none;
  z-index: 9999;
}
```

## Spacing

- **Base unit:** 8px
- **Density:** Comfortable — generous whitespace for reading text, tighter for data tables. Not a dashboard, not a notebook. Notion is the reference for content width; Bear is the reference for breathing room.
- **Scale:**
  ```
  2xs   2px
  xs    4px
  sm    8px
  md    16px
  lg    24px
  xl    32px
  2xl   48px
  3xl   64px
  4xl   96px
  ```
- **Max content width:** 680px (reading-optimized, Notion-style)
- **Sidebar width:** 220px (when present)

## Layout

- **Approach:** Grid-disciplined
- **Structure:** Sidebar (220px) + main content area (max 680px) for multi-panel screens. Single column for narrative screens (Your Story, Profile).
- **Grid:** 12 columns, 24px gutter, 32px page margin
- **Border radius scale:**
  ```
  sm    4px    — badges, tags, small buttons
  md    6px    — buttons, inputs, small cards
  lg    8px    — cards, panels
  xl    12px   — modals, drawers, app window chrome
  full  9999px — pills, avatars
  ```

---

## Motion

- **Approach:** Intentional — only transitions that aid comprehension. This is a thinking tool, not a toy.
- **Easing:** `ease-out` for enters, `ease-in` for exits, `ease-in-out` for state transitions
- **Duration:**
  ```
  micro   50–100ms   — hover state color changes
  short   150ms      — button press, toggle, badge state
  medium  250ms      — screen fade-in, panel slide
  long    400ms      — modal, drawer
  ```
- **Anti-patterns:** No bounce, no spring physics, no scroll-triggered choreography, no staggered list animations. If something moves, it should move with purpose.

---

## Anti-patterns (never do these)

- Purple or violet gradients as accent — Obsidian's territory
- Colored icons in colored circles — AI slop pattern
- Gradient buttons — does not match the aesthetic
- Centered everything with uniform spacing — too generic
- Generic stock-photo hero sections — not applicable to CLI, flag it for web layer
- Cold #000000 black backgrounds — always use warm near-black (#141412)
- Stark #FFFFFF white on dark — always use warm parchment (#C8C3BA) for primary text
- Heavy drop shadows — use subtle 1px border + minimal box-shadow only

## Decisions Log

| Date | Decision | Rationale |
|------|----------|-----------|
| 2026-03-19 | Fraunces for display | Literary serif gives artifact. a point of view nobody in the Bear/Obsidian space has. Risk: slightly less "software," more "publication." Worth it. |
| 2026-03-19 | Instrument Sans replaces DM Sans | DM Sans has insufficient weight contrast on dark backgrounds. Instrument Sans is crisper at bold weights. |
| 2026-03-19 | Amber/copper accent (#C17F3E) | Bear owns teal-green, Obsidian owns purple. Amber reads archaeological, warm, discovered. Ownable. |
| 2026-03-19 | Primary text #C8C3BA not #E8E6E1 | #E8E6E1 was too white/stark. #C8C3BA is warm parchment — archival, not terminal. |
| 2026-03-19 | Grain texture on dark surfaces | Makes the product feel physical and archival at 3–5% opacity. Adds analog warmth without any visible decoration. |
| 2026-03-19 | Lucide Icons for web layer, emoji for CLI | Emoji is terminal-native and appropriate for rich/questionary CLI. Lucide is the right icon system for any future web or GUI layer. |
| 2026-03-19 | Dark mode default | The audience lives in dark terminals. Default to their environment. |
